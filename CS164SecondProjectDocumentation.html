Description:

GPA Calculator is a JavaScript application that calculates a user’s grade point average given classes’ grades and credit weight. The user's’ grades are inputted in class grade percentages, from 0 to 100 percent.

For Users:

There are eight rows: each for one class. Grades and credits do not have to start at class 1, and can start anywhere, as long as grades and credits are in their corresponding columns and respective class rows. Only numerical values are allowed for both grades and credits. For grades, the user may input any numerical value, including values greater than 100 and less than 0. The grade point average is out of 4.0, but can be less than 0 and greater than 4.0 due to the application accepting all possible decimal grade values.

For Developers:

GPA Calculator is written in an HTML5 file, using JavaScript and CSS. First, the user enters the values into the needed textareas provided. Not all textareas need to be filled. A “Reset” button is also added that resets for a user’s convenience. Then the user clicks the button labeled “Calculate GPA” which outputs a line of text below the two buttons that reads “GPA = value” where value represents the grade point average out of 4.0. After this, the program checks for every row if only one of the two textareas is a number. If so, the program will set the value of the non-numerical textarea to “Insufficient information.” and notify the user below the two buttons in text telling them “One or more classes have insufficient information in one of their fields. These classes have been omitted from the calculations.”. This is meant so that if there exists any insufficient information, the program will completely ignore it. If both textareas contain no value, then the program completely ignores it and removes the error message. If the program completely ignores any classes, it sets the respective class grade to 0 and credit to 0 so that it will not affect the grade point sum and credit sum. The only case the program reads values if both textareas contain numerical values. Once the program reads the values, it adds the class credits to the credit sum and the grade point (grade with weight of the class credit) to the grade points sum. Once the credits and grade points are retrieved, the program outputs the GPA below the two buttons.
